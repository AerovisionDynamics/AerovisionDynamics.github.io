## Local development

1. Use Node.js 20.11.1 (or any Node version >= 20.9.0). If you use `nvm`, simply run `nvm use` to pick up the version from `.nvmrc`.
2. Install dependencies with `npm ci`.
3. Run `npm run dev` to start the local server, or `npm run build` to reproduce the production build.
